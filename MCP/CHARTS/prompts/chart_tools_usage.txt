Charting Tool Usage Guidelines:

- Only use the following chart tools: generate_line_chart, generate_bar_chart, generate_pie_chart.
- For line and bar charts, transform your tabular data into the required flat array format:
  [
    {"group": "SeriesName", "time": "XValue", "value": YValue},
    ...
  ]
- For pie charts, use:
  [
    {"label": "LabelName", "value": Value},
    ...
  ]
- Always set "title" as a string, not an object.
- Do not invent tool names or schemas.
- If unsure, default to generate_line_chart with the above schema.
